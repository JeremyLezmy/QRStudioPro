# Shared Resources

Ce dossier contient les ressources partagées entre applications du monorepo.

## assets/

- `app_icon.png`, `app_icon.ico`, `app_icon.icns`
- logos intégrés (`logos/`)

Le build desktop PyInstaller embarque `shared/assets` dans le bundle runtime `assets/`.

## presets/

Dossier pour presets JSON partagés entre les apps.
